# DPDK-TCP

A user space TCP/IP stack library with DPDK-supported kernel-bypass.

* [x] socket-like API
* [x] arp reply
* [ ] routing table
* [x] ip/icmp reply
* [x] tcp state machine
* [x] tcp window management
* [ ] tcp congestion control
* [ ] tcp keep alive
* [ ] tcp timeout retransmit
* [x] A thread-safe queue between user/app thread and DPDK thread
